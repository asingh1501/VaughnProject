# Math Sprint - Vaughn Elementary

A bright, classroom-friendly math practice game for 2nd graders. Students choose Addition, Subtraction, Multiplication, Division, or Mixed Mode and answer as many problems as possible in 60 seconds.

## Features

- Five modes: Addition, Subtraction, Multiplication, Division, and Mixed Mode
- Multiplication and division use simple story-style prompts to show groups and sharing
- 60-second timer with a progress bar
- Score, total answered, correct answers, and accuracy tracking
- Missed-problem review at the end of each round
- Best score saved in the browser for each mode
- Print-friendly results screen
- Large buttons, keyboard support, and a number keypad

## How to Run

Open `index.html` in any modern browser.

You can also serve the folder locally:

```bash
cd Vaughn_Elementary_Project
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Files

- `index.html` - page structure and game screens
- `styles.css` - responsive, colorful classroom design
- `script.js` - game timer, scoring, problem generation, and input logic

No login, database, or external dependencies are required.
